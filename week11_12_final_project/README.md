# Weeks 11–12: Final Project

## Objective

Choose a realistic AI project, scope it honestly, build a first useful version, and explain your design decisions clearly.

## Required Videos

- Watch one short demo of a small AI product and analyze its scope
- Optional: watch a builder talk about shipping simple versions first

## Tasks

1. Read `project_ideas.md`.
2. Pick one project or adapt a new one with similar scope.
3. Copy or customize the `final_project_template`.
4. Build the smallest useful version in Week 11.
5. Polish, test, and demo it in Week 12.

## Deliverables

- A scoped project plan
- A working local prototype
- A short demo or walkthrough
- A short retrospective on what changed and what you learned

## Checkpoint Questions

- Who is the project for?
- What is the smallest version that is still useful?
- What will you deliberately leave out?
- What feedback from the course should shape the final build?
